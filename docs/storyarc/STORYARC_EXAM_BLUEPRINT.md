# StoryArc Exam Blueprint

Assessment design authority for the 18 Exam Lab items. This blueprint precedes and governs Step 4 item generation. All content is original. TOEIC-style items imitate task *formats* generically; no official TOEIC questions are reproduced and no ETS affiliation is implied or claimed (learner-facing disclaimer required on every TOEIC-style set). No CEFR or score-conversion claims are made.

## Assessment categories (all four required categories ship per grade program)

| Category | Purpose | Format source |
| --- | --- | --- |
| School Exam Practice | Semester-exam readiness aligned to Phase E/F text types | Indonesian school-exam conventions (descriptive, generic) |
| TOEIC-style Practice | Workplace-English test familiarity, SMK/career relevance | Original items in TOEIC-like task formats |
| Interview | Spoken evidence; school-to-work/study transition | StoryArc rubric, transcript-based |
| Mock Exam | Integrated graded checkpoint | Blend of the above |

## Evidence rules (binding, per Codex mastery architecture)

1. Every scored item declares: objective key(s), skill key(s), rubric reference, and evidence intent (ASSESSED_EVIDENCE or RECALL_EVIDENCE). ≥80% of scored items must trace to journey-matrix objectives; the remainder must still declare skills.
2. Selected-response items: answer key + per-option rationale (why wrong options are wrong, using the dialogue-rules dimensions where applicable). Explanation quality is a validation gate.
3. Speaking/writing items: published analytic rubric per criterion — relevance, grammar, vocabulary, naturalness, clarity of expressed idea (speaking); task fulfillment, organization, grammar, vocabulary, register (writing). Transcript-based only; **no pronunciation scoring** (gap G3). Whether AI scoring may be summative is an open decision — until resolved, speaking/writing items are designed to support the manual-review path proven by the English interview pattern.
4. Mock exams include ≥2 explicit RECALL_EVIDENCE sections drawing on expressions/objectives from earlier in the grade (vocabulary strategy recall links).
5. No item depends on Story Mode plot knowledge to be answerable; StoryArc settings may flavor contexts, but every stem is self-contained (assessment validity for learners who lag in Story Mode).

## Interview design ladder

ex-g10-05 (talk about yourself) → ex-g11-04 (talk about experience with others) → ex-g12-04 (perform under real-transition stakes, dual SMA/SMK-relevant variants). Each reuses and extends prior criteria so speaking evidence is longitudinal. John hosts interview practice framing (formative rehearsal mode) distinct from the assessed attempt (per learning architecture: John free conversation never alters mastery).

## Administration constraints for generation (Step 4)

- Item counts above are targets; final counts fixed at generation and reported in the generation report.
- Every audio item requires script + audio asset reference + caption/transcript (gap G4).
- Timed multi-section mock exams depend on gap G5 (sectioned timed runtime); if unresolved, mocks degrade gracefully to untimed sections without content changes.
- All items validate against the actual Codex content schema/import contract — which does not yet exist in the repository and blocks Step 4.
