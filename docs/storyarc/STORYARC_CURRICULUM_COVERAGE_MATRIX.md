# StoryArc Curriculum Coverage Matrix

StoryArc is curriculum-mapped, not an official government textbook.

**CURRICULUM SOURCE GAP (governing note):** project materials contain the approved mapping *labels* only — Grade 10 → Phase E, Grades 11–12 → Phase F. No verified Capaian Pembelajaran text, CP identifiers, ATP sequences, or SMK vocational-English supplements are present in the repository. Therefore this matrix maps content to **descriptive competency areas** consistent with the approved phase labels. No official identifiers are cited. Before release marketing or reporting claims official alignment, the academic lead must supply verified CP text (open decision, Phase 0) and this matrix must be re-validated against it.

CEFR and TOEIC alignment are intentionally **not claimed** (CURRICULUM SOURCE GAP C4/C5 in the product challenge).

Competency-area keys are internal StoryArc taxonomy (`cpa-*`), designed to be re-mapped to verified CP references later without renaming content.

## SMA / SMK applicability

- SMA relevance: text-type and exam-oriented areas (cpa-e-desc/recount/narr, cpa-f-expo) plus school-exam practice per grade.
- SMK relevance (CURRICULUM SOURCE GAP C2 — no verified vocational supplement; designed from general phase competencies applied to vocational contexts): cpa-e-proc (sc-g10-13, sm-g10-08), cpa-f-task (sc-g11-07/08), and the entire Grade 12 work strand (cpa-f-worklit, cpa-f-prospeak, cpa-f-worklisten, cpa-f-report; internship arc sm-g12-04..06).
- Both SMA and SMK futures are assessed in ex-g12-04 (interview ships university and job variants).

## Coverage verification rules

1. Every School Core item maps to ≥1 competency area; every competency area has ≥1 item in each track (verified above).
2. Sequencing within a grade is a design proposal (CURRICULUM SOURCE GAP C3), recommended order recorded in the manifest.
3. When verified CP text arrives, each `cpa-*` key gets an authoritative reference field; content IDs do not change.
