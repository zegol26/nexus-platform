# Glossary

- Ai-chan: Nexus AI Nihongo tutor/assistant persona.
- Andromeda: PMP AI instructor persona.
- App access: Per-user entitlement stored in `AppUserAccess`.
- DCE: Dynamic Conversational English curriculum.
- FeatureUsage: Counter model used for trial/usage limits such as AI tutor and voice.
- John: English AI conversation coach persona.
- Midtrans: Payment gateway used for checkout and billing.
- Nexus Kingdoms: Gamification/battle layer tied to learning rewards.
- PlatformSetting: Database-backed operational settings, excluding secrets.
- ReadingPassage: Shared content table for reading/listening entries.
- Restricted overview trial: Public app preview that mimics UI without real AI, recording, scoring, or persistence.
- ServerRouteMetric: Route-level metric model for Fluid CPU/slow/error visibility.
- Single active session: Auth policy allowing one active non-admin session record per user.
